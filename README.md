# Bookmark Saver

A lightweight, client-side bookmark manager that lets you save, organize, and instantly open your favorite web resources through clean, interactive bookmark cards.

---

## Overview

In day-to-day development and browsing, important links quickly become scattered across tabs, notes, and history. Managing multiple resourcesвЂ”fonts, SVG libraries, tools, documentation sitesвЂ”often leads to clutter and poor recall of URLs.

Bookmark Saver solves this by providing a simple, visual system to store and access links in one place.

Instead of remembering URLs or digging through browser bookmarks, users can assign a name (alias) to any link and instantly launch it later through an intuitive card-based interface.

Everything runs directly in the browser using HTML, CSS, and JavaScriptвЂ”making it fast, lightweight, and easy to use.

---

## Features

### Visual Bookmark Cards

Saved links are displayed as clean, interactive cards instead of plain text entries, improving readability and usability.

### Instant Link Access

Clicking a bookmark card opens the saved URL directly in a new tab for a smooth browsing experience.

### Alias-Based Organization

Each URL can be assigned a custom name, making it easier to remember and organize resources meaningfully.

### One-Click Removal

Each bookmark includes a built-in remove option, allowing users to delete entries instantly when no longer needed.

### Input Validation

Ensures both name and URL fields are provided before creating a bookmark, preventing incomplete entries.

### Fully Responsive UI

Optimized layout that adapts seamlessly across desktop, tablet, and mobile devices.

## Usage

- Open the application in your browser
- Enter a name (alias) for the bookmark
- Paste the URL of the website
- Click Create Bookmark
- A new bookmark card will appear instantly
- Click the card to open the saved link in a new tab
- Use the remove button to delete any bookmark

The interface is designed to be minimal, fast, and distraction-free.
